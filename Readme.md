# AnalytixBot

AnalytixBot is a production-ready **AI Workflow Automation** platform designed to streamline customer qualification and routing processes. It leverages advanced AI flows to capture leads, enrich customer details, score intent, and automatically route high-value opportunities to the right team.

Powered by ![FastAPI](https://img.shields.io/badge/-FastAPI-009688?style=flat&logo=fastapi&logoColor=white) and ![LangGraph](https://img.shields.io/badge/-LangGraph-7E3AED?style=flat&logo=semanticweb&logoColor=white), it creates dynamic automation pipelines capable of replacing repetitive manual steps.

AnalytixBot comes with a Admin Panel for monitoring workflow executions, chat responses, and overall lead funnel performance ensuring faster pipeline creation and improved conversion outcomes. Deployed on ![Render](https://img.shields.io/badge/-Render-46E3B7?style=flat&logo=render&logoColor=white), it is scalable, cloud-ready, and optimized for real-world enterprise environments.

## Run Locally

### Clone the repository

```bash
git clone https://github.com/Ai-firelab/AnalytixBot.git
cd AnalytixBot
```
### Create virtual environment
```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```
### Install dependencies
```bash
pip install -r requirements.txt
```
